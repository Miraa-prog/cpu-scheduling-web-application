# 💻 CPU Scheduling Web Application

## 📌 Overview
A full-stack web application that simulates CPU scheduling algorithms used in Operating Systems.  
The system allows users to input processes and visualize scheduling results with detailed calculations.

---

## 🎯 Features
- Process input system
- Real-time scheduling simulation
- Calculation of:
  - Waiting Time
  - Turnaround Time
  - Completion Time
- Visualization of scheduling results
- Interactive web interface

---

## ⚙️ Scheduling Algorithms Implemented
- First Come First Serve (FCFS)
- Priority Scheduling (Preemptive & Non-Preemptive)
- Round Robin (RR)
